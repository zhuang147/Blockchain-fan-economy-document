# Blockchain-fan-economy
本研究聚焦於演唱會票務與粉絲經濟所面臨的信任危機與價值錯置問題，提出以單一團體與其粉絲為核心之整合性解決方案。透過建構一套基於 Web2.5 架構的區塊鏈系統，將票務、周邊商品與粉絲互動機制進行整合。系統導入智慧合約以實現受控票務機制，在防止黃牛與詐騙的同時，保留粉絲間合理讓票的彈性；並結合 NFT 與 NFC 技術，為實體周邊建立可驗證的數位身分，以確保商品真實性與價值回流。此外，透過粉絲任務卡與數位足跡紀錄，將觀看演唱會、購買專輯與參與活動等行為轉化為長期會員價值，強化粉絲黏著度。整體系統採用「後端鏈上化、前端無感化」之設計，結合法幣支付與託管錢包，降低使用門檻並避免虛擬貨幣波動風險，打造兼具安全性、可用性與長尾效益之粉絲經濟生態系。
